# Linux 基本開発環境構築スクリプト集 だにゃ

このリポジトリは、Linux環境を新しく作ったり、別のPCに引っ越したりする時に、ボクたちの作業環境を「一瞬で、いつもの最高の状態」に再現するための秘伝の書だにゃ。

---

## 目的：なぜこれが必要なのかにゃ？

新しいマシンを触るたびに、手動でコマンドを何十回も叩いて環境を作るのは時間の無駄だし、何より「設定のやり忘れ」が起きるのが一番怖いにゃ。

このスクリプト集は、以下の役割を担っているにゃ：

1. **自動化による時短**: 複雑な依存関係やドライバの導入を、スクリプト一つで安全に完了させるにゃ。
2. **環境の同一化**: どのPCを使っても、同じ操作感、同じ設定で作業ができる「安心感」を提供するにゃ。
3. **ナレッジの蓄積**: 「どうやって設定したっけ？」という手順をドキュメントとスクリプトとして残しておくことで、未来の自分を助けるための資産にするにゃ。

---

## 役割の分類だにゃ

大きく分けて3つの軸で構成されているにゃ。

* **System（土台作り）**:
キーボードの配列やGPUドライバなど、PCという「箱」を正しく動かすための根本的な設定だにゃ。
* **Apps（道具箱）**:
DockerやVim、Gitなど、ボクたちが開発という「戦い」に出るために必要な武器を揃える場所だにゃ。
* **Scripts（便利ツール）**:
日々のバックアップやファイル変換など、ちょっとした「面倒くさい」を解決する小道具たちが詰まっているにゃ。

---

## 使い方への想いだにゃ

ただスクリプトを走らせるだけじゃなく、各ディレクトリにあるドキュメント（`doc_xxx.md`）を読むことで、「なぜこの設定が必要なのか」という文脈も理解できるようにしてあるにゃ。

これさえあれば、どんな過酷な環境（新規インストール直後のOSなど）に放り出されても、一瞬で「いつもの城」を築き上げることができるにゃ！
